# TextAnalysForBusiness
[Датасет](https://www.kaggle.com/datasets/tarkkaanko/amazon)
REST API который получает на вход текст (отзыв) а на выход отдает всю нужную для бизнеса информацию:
1) Сентиментальный анализ(Положительный/Нейтральный/Отрицательный отзыз)
2) Тему отзыва
3) Саммари отзыва

Основная модель которая использовалась TF-IDF
Сентиментальная модель использует ансамбль из 3 логрегов, каждая из которых обучена на специфичном tf-idf

```
docker build -t nlp .
docker run -p 8000:8000 nlp
```
Дальнейшее развитие:
1) Обучение на более мастштабном датасете из разных областей ecommerce 
2) Обучение на русскоязычном датасете
3) Внедрение в компании и finetuning для спецефичных задач
4) Обучение определять топик на размеченных данных
